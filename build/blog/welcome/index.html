We're excited to launch the official KeyCase documentation site!

This site is your go-to resource for learning how to use KeyCase — from getting started to advanced administration.

What You'll Find Here​

  • Getting Started — Sign in, navigate the app, and take the welcome tour.
  • Core Concepts — Understand keywords, flows, agents, and executions.
  • User Guides — Step-by-step instructions for everyday tasks.
  • Administration — Manage users, roles, agents, and organizations.
  • Reference — Glossary and keyboard shortcuts.

Stay Updated​

We'll be using this blog to share:

  • Release notes and changelogs
  • Tips and best practices
  • Feature announcements

Bookmark this site and check back regularly for updates.
