# User Management

Organization administrators can manage users — inviting new members, updating roles, and removing access.

## Viewing Users

Navigate to **Administration > Users** to see all users in your organization. The list shows each user's name, email, role, and status.

## Inviting Users

1. Click **Invite User**.
2. Enter the user's email address.
3. Select an organization-level role.
4. Send the invitation.

The user will receive an email with instructions to join the organization.

## Editing Users

Click on a user to view their profile, where you can:

- Change their organization-level role.
- View which projects they belong to.
- Deactivate or reactivate their account.

## Removing Users

To remove a user from the organization:

1. Open the user's profile.
2. Click **Remove from Organization**.
3. Confirm the action.

This removes the user from the organization and all its projects. Their KeyCase account remains active if they belong to other organizations.
