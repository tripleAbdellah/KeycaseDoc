# Keyboard Shortcuts

KeyCase supports keyboard shortcuts to help you work faster.

## Global Shortcuts

| Shortcut | Action |
|---|---|
| `Ctrl/Cmd + K` | Open the command palette / quick search |
| `Ctrl/Cmd + S` | Save the current resource |

## Flow Designer

| Shortcut | Action |
|---|---|
| `Ctrl/Cmd + Z` | Undo |
| `Ctrl/Cmd + Shift + Z` | Redo |
| `Delete` / `Backspace` | Delete selected step |
| `Ctrl/Cmd + D` | Duplicate selected step |

## Navigation

| Shortcut | Action |
|---|---|
| `Ctrl/Cmd + 1` | Go to Projects |
| `Ctrl/Cmd + 2` | Go to Keywords |
| `Ctrl/Cmd + 3` | Go to Flows |
| `Ctrl/Cmd + 4` | Go to Executions |

:::note
Keyboard shortcuts may vary depending on your operating system and browser. The shortcuts listed above use `Ctrl` for Windows/Linux and `Cmd` for macOS.
:::
