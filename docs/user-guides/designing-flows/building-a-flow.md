# Building a Flow

Flows combine multiple keywords into a complete automation sequence. This guide walks you through building one from scratch.

## Creating a New Flow

1. Navigate to the **Flows** section within your project.
2. Click **New Flow** (or right-click a folder and select **New Flow**).
3. Enter a name for the flow.
4. The flow designer opens with an empty canvas.

## Adding Keyword Steps

1. In the flow designer, click **Add Step** or use the keyword panel.
2. Search for the keyword you want to add.
3. Select the keyword (or a specific instance of it).
4. The keyword appears as a step in the flow sequence.

## Ordering Steps

Steps execute in the order they appear. You can:

- **Drag to reorder** — Move steps up or down in the sequence.
- **Delete steps** — Remove steps you no longer need.
- **Duplicate steps** — Copy an existing step with its configuration.

## Configuring Step Parameters

For each step, you can:

- Set **static values** for input parameters.
- **Connect** input parameters to outputs from previous steps.
- Mark parameters as using **environment variables**.

See [Connecting Parameters](/docs/user-guides/designing-flows/connecting-parameters) for more on data wiring.

## Saving

Click **Save** to persist your flow. The flow is now ready to be included in an execution plan.
