# Creating Keywords

Keywords are the reusable building blocks of your automation. This guide walks you through creating your first keyword.

## Steps

1. Navigate to **Keywords** in the sidebar (within a project).
2. Click **Create Keyword**.
3. Enter the keyword details:
   - **Name** — A clear, descriptive name (e.g., `Open Login Page`).
   - **Description** — What this keyword does.
4. Click **Create**.

## After Creation

Once created, you can:

- **Add parameters** — Define the inputs and outputs for this keyword. See [Keyword Parameters](/docs/user-guides/working-with-keywords/keyword-parameters).
- **Create instances** — Configure specific parameter values for reuse. See [Keyword Instances](/docs/user-guides/working-with-keywords/keyword-instances).
- **Use in flows** — Reference this keyword as a step in any flow.

## Naming Conventions

Use descriptive, action-oriented names for keywords:

- `Click Submit Button`
- `Verify User Is Logged In`
- `Fetch API Response`

This makes flows easier to read and understand.
