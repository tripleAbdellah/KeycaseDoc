# Creating a Project

Projects are the top-level container for organizing your automation work in KeyCase. Each project holds its own keywords, flows, environments, and executions.

## Steps to Create a Project

1. Navigate to **Projects** in the sidebar.
2. Click the **Create Project** button.
3. Fill in the project details:
   - **Name** — A descriptive name for the project.
   - **Description** — Optional context about the project's purpose.
4. Click **Create** to finalize.

Your new project will appear in the projects list and you can begin adding keywords, flows, and team members.

## Project Settings

After creation, you can configure additional settings from the project's settings page:

- **General** — Update the name and description.
- **Members** — Add or remove team members and assign roles.
- **Environments** — Define environment-specific variables.
- **Import/Export** — Transfer project data between instances.
