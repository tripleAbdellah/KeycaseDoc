# Viewing Results

After an execution completes, KeyCase provides detailed results for every step.

## Execution Summary

The execution detail page shows:

- **Overall result** — Passed or Failed.
- **Duration** — Total execution time.
- **Agent** — Which agent ran the execution.
- **Environment** — Which environment was used.
- **Timestamp** — When the execution started and ended.

## Step-by-Step Results

Each keyword step in the flow is listed with its individual result:

| Status | Meaning |
|---|---|
| **Passed** | The step completed successfully |
| **Failed** | The step encountered an error |
| **Skipped** | The step was skipped (e.g., due to a prior failure) |

Click on any step to expand its details, including:

- Input and output parameter values.
- Logs and error messages.
- Execution duration.

## Execution History

All executions are listed in the **Executions** section, sorted by most recent. You can filter by status, date range, or flow name to find specific runs.
