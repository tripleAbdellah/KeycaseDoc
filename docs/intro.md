# What is KeyCase?

KeyCase is a **multi-tenant process and test automation platform** that lets teams design, manage, and execute automated workflows through an intuitive web interface.

## Key Capabilities

- **Keywords** — Reusable building blocks that represent individual actions or test steps.
- **Flows** — Visual compositions of keywords that define end-to-end automation sequences.
- **Agents** — Execution runtimes that run your automation flows on target environments.
- **Executions** — Run, monitor, and review automation results in real time.

## Who Is It For?

KeyCase is built for QA engineers, automation developers, and platform teams who need a structured way to create, organize, and run automation at scale — across projects and organizations.
